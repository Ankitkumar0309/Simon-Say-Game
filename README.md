Simon Says Game 🎮

A fun and interactive Simon Says Game built using HTML, CSS, and JavaScript.

📌 Overview

Simon Says is a memory-based game where players must repeat an increasingly long sequence of colors. The game tests concentration, memory, and reaction skills.

🚀 Features

- Random color sequence generation
- Interactive button animations
- Score tracking based on levels
- Game-over detection
- Responsive and user-friendly interface
- Restart and replay functionality

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

🎯 How to Play

1. Press any key to start the game.
2. Watch the color that flashes.
3. Click the same color.
4. As the game progresses, remember and repeat the entire sequence.
5. The game ends if you click the wrong color.
